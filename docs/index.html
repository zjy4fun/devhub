功能模块

📦

Git 配置

查看配置、健康检查、修改用户名邮箱、默认分支、pull 策略、GPG 签名提醒。

🔐

SSH 配置

密钥扫描、权限检查、agent 状态、主机配置、一键修复权限、生成密钥、测试连接。

🔑

环境变量

变量溯源（哪个文件哪一行）、重复检测、PATH 拆解分析、敏感值遮罩。

💚

Node.js 生态

检测 Node/npm/pnpm/yarn/bun/nvm/fnm，npm registry 一键切换官方/淘宝镜像。

📥

工具安装

22+ 常用开发工具引导安装，内置中国镜像源，支持官方/镜像一键切换。

🇨🇳

中国镜像

npm、Homebrew、pip、rustup、Go、Docker、GitHub Release 镜像地址内置。

🇨🇳 为中国开发者优化

内置主流镜像源地址，npm/pip/Homebrew/Go/Rust 一键切换，不再为网络发愁。
